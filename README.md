POMO — Pomodoro Task Manager
A minimal, focused productivity app combining a Pomodoro timer with a full task management system. Built with React and Vite.

Features
Timer

25 / 5 / 15 minute Pomodoro cycles (focus, short break, long break)
Fully adjustable durations per mode via sliders and step buttons
Circular progress ring with smooth animation
Session counter tracking rounds of four
Optional ambient tick sound via Web Audio API (no external dependencies)

Task Management

Add tasks with priority levels — critical, high, medium, low
Set deadlines with live countdowns (overdue detection included)
Assign estimated pomodoro count per task and track progress
Tag tasks by energy level (high / medium / low) for smarter scheduling
Custom tags and notes per task
Filter by priority, sort by deadline or pomodoros completed
One-click focus routing — select a task and jump straight to the timer

Unique Features

Energy-based task suggestion — surfaces high-energy tasks when you're fresh
Per-task pomodoro estimation — set how many sessions a task should take and watch it fill
Inline focus routing — no friction between picking a task and starting the timer
Priority completion bars in stats — see what percentage of each priority tier is done
Activity heatmap — 28-day session history at a glance

Stats

Sessions completed today
Total minutes focused
Task completion rate
Daily streak tracker
Pomodoros logged per task


Tech Stack

React 18 — UI and state management
Vite — build tool and dev server
Web Audio API — ambient sound, zero dependencies
CSS-in-JS — fully inline styles, no external CSS libraries
